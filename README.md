# CSharpEducation601

> An educational, multi-database supported WinForms application built on a layered architecture

This project is a sample desktop application developed within the scope of the **C# Education Camp**. The main purpose of the project is to practically learn and reinforce modern software development approaches such as the **C# programming language**, **layered architecture**, **object-oriented programming (OOP)**, and **multi-database integration**.
